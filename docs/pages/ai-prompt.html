AI Prompt

Use this prompt with ChatGPT (or any LLM) to generate a custom scoreboard overlay that is compatible with Fly Scoreboard‚Äôs templating runtime.

Keep the output strictly to markup/CSS as requested. The runtime that performs template replacement is already provided by script.js.

Copy/paste prompt

ROLE: You are a senior frontend engineer building an OBS Browser Source scoreboard overlay.

TARGET SYSTEM: Fly Scoreboard plugin overlay runtime.
- Data comes from plugin.json (same folder as index.html).
- The runtime replaces {{ ... }} placeholders in text and attributes.
- Conditional visibility uses fs-if="boolean expression".
- No loops are available; author fixed slots and guard them with fs-if.
- Prefer swap-safe helpers provided by the runtime:
  - team_x / team_y (left/right teams based on swap_sides)
  - fields_xy[i].x / fields_xy[i].y (left/right values for custom_fields)
  - timers[i].mmss and timers[i].label (timer display)

OUTPUT REQUIREMENTS:
1) Output ONLY the scoreboard layout markup (no <html>, no <head>), unless explicitly asked.
2) Do NOT include external libraries, frameworks, or CDN scripts.
3) Use semantic, class-based HTML that can be styled with plain CSS.
4) Include placeholders for:
   - team_x.title, team_x.subtitle, team_x.logo
   - team_y.title, team_y.subtitle, team_y.logo
   - at least 3 custom field rows using fields_xy[0..2]
   - at least 2 timers using timers[0..1]
   - at least 2 single stats using single_stats[0..1]
5) Add fs-if guards for every optional index:
   - fs-if="fields_xy[2] && fields_xy[2].visible"
   - fs-if="timers[1] && timers[1].visible"
   - fs-if="single_stats[1] && single_stats[1].visible"
6) Provide a short CSS block (optional) that matches a modern broadcast style.

CONTEXT (Data shape examples):
- team_x.title (string), team_x.logo (string path), team_x.color (hex string)
- fields_xy[i].label (string), fields_xy[i].x (number/string), fields_xy[i].y (number/string), fields_xy[i].visible (bool)
- timers[i].label (string), timers[i].mmss (string), timers[i].visible (bool)
- single_stats[i].label (string), single_stats[i].value (string/number), single_stats[i].visible (bool)

NOW: Generate the layout for a compact lower-third scoreboard aligned top-left, with a clean hierarchy and good readability.

Example placeholders you can reference

  • {{show_scoreboard ? 'board' : 'board is-hidden'}} (attribute ternary)
  • {{team_x.title}}, {{team_y.title}}
  • {{fields_xy[0].x}}, {{fields_xy[0].y}}
  • {{timers[0].mmss}}, {{timers[1].label}}
