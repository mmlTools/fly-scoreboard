Overview

Fly Scoreboard is an OBS Studio plugin that writes a live plugin.json state file and ships a lightweight overlay runtime that renders your scoreboard by replacing template placeholders ({{…}}) in HTML.

What you get

  • A dock UI to manage teams, scores, timers, and single stats.
  • A ready-to-use overlay (index.html, style.css, script.js).
  • A small templating engine: write {{home.title}}, {{timers[0].mmss}}, etc.

How it works

  1. The plugin updates plugin.json (your “source of truth”).
  2. The overlay fetches plugin.json once per second (no-cache).
  3. An animation loop renders timers smoothly and applies template replacements every frame.

Quick start

  1. Install the plugin.
  2. Add a Browser Source in OBS and point it to the overlay index.html.
  3. In the Fly Scoreboard dock, select that Browser Source (so the plugin can keep it in sync).
  4. Customize teams, add custom fields (scores), set timers, and go live.
Tip: Keep index.html, style.css, script.js and plugin.json in the same folder. The default overlay runtime fetches plugin.json by a relative path.
