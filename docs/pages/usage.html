How to Use

Operate the scoreboard from the dock during a live show: update teams, increment scores, and control timers.

Typical live workflow

  1. Select your Browser Source in the dock (the overlay target).
  2. Set team names, subtitles, and logos.
  3. Update scores via custom fields.
  4. Start/stop timers and switch labels as needed.
  5. Use hotkeys (if configured) to control common actions quickly.
  6. Use a WebSocket remote controller for stream decks, phones, or automation tools.

Understanding the main data groups

Teams

Team identity and theme values used by the overlay.

  • home, away
  • title, subtitle, logo, color
  • swap_sides changes how they map to team_x/team_y

Custom fields (scores)

You can add multiple score rows (e.g., score, fouls, shots).

  • custom_fields[] stored as home/away pairs
  • Overlay-safe mapping via fields_xy[]
  • Per-field visible flag

Timers

Multiple timers supported. The overlay computes mmss live.

  • timers[] with mode, running, remaining_ms
  • Use timers[i].mmss in templates
  • Use fs-if with timers[i].visible

Single stats

Single value indicators like PERIOD, ROUND, SET, etc.

  • single_stats[] label/value/visible
  • Reference directly (e.g. {{single_stats[0].value}})

Remote control commands

Connect to ws://127.0.0.1:4457 and send JSON commands. The plugin broadcasts updated state after every accepted command. 

{"type":"get_state"}
{"action":"bump_score","index":0,"side":"home","delta":1}
{"action":"set_score","index":0,"side":"away","value":12}
{"action":"timer_start","index":0}
{"action":"timer_pause","index":0}
{"action":"timer_reset","index":0}
{"action":"swap"}
{"action":"show_scoreboard","value":true}
{"action":"load_template","name":"Soccer Lower Third"}
If you want a fully custom look, copy the overlay folder and edit index.html + style.css. Keep script.js unless you are intentionally changing the runtime.
