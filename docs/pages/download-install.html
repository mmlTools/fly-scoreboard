Step-by-step installation and verification inside OBS Studio.

  • Download the latest release ZIP from the official page above.
  • Verify you downloaded the correct build for your OBS version and OS.

2) Install (manual)

  1. Close OBS Studio completely.
  2. Extract the ZIP.
  3. Copy the plugin files into your OBS installation folders.

Windows (typical)

Copy plugin DLLs into:

%ProgramFiles%\obs-studio\obs-plugins\64bit\

Copy the plugin data folder into:

%ProgramFiles%\obs-studio\data\obs-plugins\fly-scoreboard\

Portable OBS / custom paths

If you run OBS as portable or installed elsewhere, copy into the equivalent:

<OBS_ROOT>\obs-plugins\64bit\
<OBS_ROOT>\data\obs-plugins\fly-scoreboard\
Conflicts: If you previously used older builds with different plugin names, remove/rename the old DLL to avoid double-loading or hotkey collisions.

3) Verify in OBS

  1. Start OBS Studio.
  2. Open Tools ‚Üí Fly Scoreboard (or the dock menu entry).
  3. If the dock loads, the plugin is installed.

Included overlay files

The plugin ships an overlay folder containing:

  • index.html ‚Äì markup with template placeholders
  • style.css ‚Äì visual styling
  • script.js ‚Äì the runtime (fetch + templating + timer render loop)
  • plugin.json ‚Äì sample data shape (the plugin overwrites this at runtime)
You can copy the overlay folder anywhere (for theming/versioning). The only strict requirement is that your overlay runtime can access the live plugin.json.
